# UAV-Assisted NB-IoT Coverage Simulation over Melbourne CBD

## Overview
This project simulates the downlink coverage performance of a UAV-assisted Narrowband IoT (NB-IoT) access network over Melbourne CBD.

The simulation places a UAV-mounted NB-IoT base station near the intersection of Bourke Street and Elizabeth Street and evaluates coverage at different UAV altitudes. The analysis considers received power, signal-to-noise ratio (SNR), block error rate (BLER), and coverage percentage.

The project was developed as part of Wireless Sensor Networks and IoT coursework and is documented here as a public technical portfolio project.

## Project Objectives
- Simulate UAV-assisted NB-IoT coverage in a dense urban environment
- Evaluate how UAV altitude affects received power and SNR
- Compare NB-IoT coverage with repetition settings NRep = 1 and NRep = 32
- Use 5% BLER as the coverage reliability threshold
- Visualise receiver grid, coverage maps, SNR maps and coverage contours
- Generate a coverage summary for different UAV heights
